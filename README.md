pop-os
===========

### Usage

```bash
curl -s https://raw.githubusercontent.com/chris-m-powell/pop-os/master/deploy.sh | sudo bash -s -- [<options>]
```
### Options

| Options | Argument                                          |
|:-------:|:--------------------------------------------------|
| -u      | Name of local Kali user                           |
| -t      | Comma-delimited list of tags (defaults to 'all')  |
| -h      | displays usage options                            |
